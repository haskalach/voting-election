# Infrastructure Layer Agent Instructions

**Purpose**: Guide AI agents working on data access, repositories, migrations, and database concerns.

**Use this file when**:

- Adding or modifying database migrations
- Creating or updating repositories
- Configuring Entity Framework Core mappings
- Handling database queries and filtering
- Managing connection strings and configurations
- Implementing data seeding

---

## Infrastructure Philosophy

**Principles**:

1. **Data access abstraction**: Repositories hide EF Core details
2. **Specifications drive queries**: Complex queries use specifications
3. **Migrations track schema**: Every change version controlled
4. **Soft deletes by default**: Queries filter IsDeleted automatically
5. **Eager loading prevents N+1**: Include() used judiciously
6. **Unit of Work coordinates changes**: SaveChanges() called once per operation

## Repository Pattern

### Generic Repository Base

```csharp
namespace ElectionVoting.Infrastructure.Repositories
{
    /// <summary>
    /// Base repository providing CRUD operations and specification pattern support.
    /// All entities use this pattern for data access.
    /// </summary>
    public class RepositoryBase<T> : IRepository<T> where T : class
    {
        protected readonly ElectionVotingContext _context;
        protected readonly DbSet<T> _dbSet;

        public RepositoryBase(ElectionVotingContext context)
        {
            _context = context ?? throw new ArgumentNullException(nameof(context));
            _dbSet = context.Set<T>();
        }

        // ✅ Create
        public async Task<T> AddAsync(T entity)
        {
            if (entity == null)
                throw new ArgumentNullException(nameof(entity));

            await _dbSet.AddAsync(entity);
            return entity;
        }

        // ✅ Read single by ID
        public async Task<T> GetByIdAsync(int id)
        {
            return await _dbSet.FindAsync(id);
        }

        // ✅ Read all
        public async Task<IEnumerable<T>> ListAsync()
        {
            return await _dbSet.ToListAsync();
        }

        // ✅ Read with specification (complex queries)
        public async Task<IEnumerable<T>> ListAsync(ISpecification<T> spec)
        {
            return await ApplySpecification(spec).ToListAsync();
        }

        // ✅ Read single with specification
        public async Task<T> FirstAsync(ISpecification<T> spec)
        {
            return await ApplySpecification(spec).FirstOrDefaultAsync();
        }

        // ✅ Count
        public async Task<int> CountAsync()
        {
            return await _dbSet.CountAsync();
        }

        // ✅ Count with specification
        public async Task<int> CountAsync(ISpecification<T> spec)
        {
            return await ApplySpecification(spec).CountAsync();
        }

        // ✅ Update
        public async Task<T> UpdateAsync(T entity)
        {
            if (entity == null)
                throw new ArgumentNullException(nameof(entity));

            _dbSet.Update(entity);
            return entity;
        }

        // ✅ Delete (hard delete - use carefully)
        public async Task DeleteAsync(T entity)
        {
            if (entity == null)
                throw new ArgumentNullException(nameof(entity));

            _dbSet.Remove(entity);
        }

        // ✅ Helper: Apply specification pattern
        protected IQueryable<T> ApplySpecification(ISpecification<T> spec)
        {
            return SpecificationEvaluator<T>.GetQuery(_dbSet, spec);
        }
    }
}
```

### Specification Evaluator

```csharp
namespace ElectionVoting.Infrastructure.Specifications
{
    /// <summary>
    /// Applies specification criteria to IQueryable for data access.
    /// </summary>
    public class SpecificationEvaluator<T> where T : class
    {
        public static IQueryable<T> GetQuery(IQueryable<T> inputQuery, ISpecification<T> specification)
        {
            var query = inputQuery;

            // ✅ Apply where conditions
            if (specification.Criteria != null)
            {
                query = query.Where(specification.Criteria);
            }

            // ✅ Apply ordering
            if (specification.OrderBy != null)
            {
                query = query.OrderBy(specification.OrderBy);
            }

            if (specification.OrderByDescending != null)
            {
                query = query.OrderByDescending(specification.OrderByDescending);
            }

            // ✅ Apply eager loading (Include related data)
            query = specification.Includes.Aggregate(query,
                (current, include) => current.Include(include));

            // ✅ Apply pagination
            if (specification.IsPagingEnabled)
            {
                query = query
                    .Skip((specification.PageNumber - 1) * specification.PageSize)
                    .Take(specification.PageSize);
            }

            return query;
        }
    }
}
```

## Entity Framework Configuration

### DbContext Setup

```csharp
namespace ElectionVoting.Infrastructure.Data
{
    /// <summary>
    /// Main database context for Election Voting system.
    /// Configures all entities, relationships, and constraints.
    /// </summary>
    public class ElectionVotingContext : DbContext
    {
        public ElectionVotingContext(DbContextOptions<ElectionVotingContext> options)
            : base(options) { }

        // ✅ DbSets for each entity
        public DbSet<Organization> Organizations { get; set; }
        public DbSet<Employee> Employees { get; set; }
        public DbSet<Election> Elections { get; set; }
        public DbSet<Candidate> Candidates { get; set; }
        public DbSet<Vote> Votes { get; set; }
        public DbSet<PollingStation> PollingStations { get; set; }
        public DbSet<AuditLog> AuditLogs { get; set; }

        protected override void OnModelCreating(ModelBuilder modelBuilder)
        {
            modelBuilder.HasDefaultSchema("dbo");

            // ✅ Organization configuration
            ConfigureOrganization(modelBuilder);

            // ✅ Employee configuration
            ConfigureEmployee(modelBuilder);

            // ✅ Election configuration
            ConfigureElection(modelBuilder);

            // ✅ Query filters (soft delete)
            ApplyQueryFilters(modelBuilder);
        }

        // ✅ Automatically set audit fields on SaveChanges
        public override Task<int> SaveChangesAsync(CancellationToken cancellationToken = default)
        {
            var entries = ChangeTracker.Entries();

            foreach (var entry in entries)
            {
                switch (entry.State)
                {
                    case EntityState.Added:
                        // ✅ On create, set CreatedAt/CreatedBy
                        if (entry.Entity is IAuditable createdEntity)
                        {
                            createdEntity.CreatedAt = DateTime.UtcNow;
                            // CreatedBy set by application service with user ID
                        }
                        break;

                    case EntityState.Modified:
                        // ✅ On update, set ModifiedAt/ModifiedBy
                        if (entry.Entity is IAuditable modifiedEntity)
                        {
                            modifiedEntity.ModifiedAt = DateTime.UtcNow;
                            // ModifiedBy set by application service with user ID
                        }
                        break;
                }
            }

            return base.SaveChangesAsync(cancellationToken);
        }

        private void ConfigureOrganization(ModelBuilder modelBuilder)
        {
            var builder = modelBuilder.Entity<Organization>();

            builder.HasKey(o => o.OrganizationId);

            builder.Property(o => o.OrganizationName)
                .IsRequired()
                .HasMaxLength(200);

            builder.Property(o => o.ManagerEmail)
                .IsRequired()
                .HasMaxLength(256);

            builder.Property(o => o.AdminFirstName)
                .IsRequired()
                .HasMaxLength(100);

            builder.Property(o => o.AdminLastName)
                .IsRequired()
                .HasMaxLength(100);

            // ✅ Indexes for performance
            builder.HasIndex(o => o.ManagerEmail).IsUnique();
            builder.HasIndex(o => o.IsDeleted);

            // ✅ Relationships
            builder.HasMany(o => o.Employees)
                .WithOne(e => e.Organization)
                .HasForeignKey(e => e.OrganizationId)
                .OnDelete(DeleteBehavior.Restrict);

            builder.HasMany(o => o.Elections)
                .WithOne(el => el.Organization)
                .HasForeignKey(el => el.OrganizationId)
                .OnDelete(DeleteBehavior.Restrict);
        }

        private void ConfigureEmployee(ModelBuilder modelBuilder)
        {
            var builder = modelBuilder.Entity<Employee>();

            builder.HasKey(e => e.EmployeeId);

            builder.Property(e => e.FirstName)
                .IsRequired()
                .HasMaxLength(100);

            builder.Property(e => e.LastName)
                .IsRequired()
                .HasMaxLength(100);

            builder.Property(e => e.Email)
                .IsRequired()
                .HasMaxLength(256);

            // ✅ Unique constraint
            builder.HasIndex(e => new { e.OrganizationId, e.Email })
                .IsUnique()
                .HasFilter("[IsDeleted] = 0");  // ✅ Only on non-deleted

            builder.HasIndex(e => e.IsDeleted);
        }

        private void ConfigureElection(ModelBuilder modelBuilder)
        {
            var builder = modelBuilder.Entity<Election>();

            builder.HasKey(e => e.ElectionId);

            builder.Property(e => e.ElectionName)
                .IsRequired()
                .HasMaxLength(200);

            builder.Property(e => e.Status)
                .HasConversion<int>();  // ✅ Store enum as int

            builder.HasIndex(e => new { e.OrganizationId, e.Status });
        }

        private void ApplyQueryFilters(ModelBuilder modelBuilder)
        {
            // ✅ Soft delete filter: exclude IsDeleted = true automatically
            foreach (var entityType in modelBuilder.Model.GetEntityTypes())
            {
                if (typeof(ISoftDeletable).IsAssignableFrom(entityType.ClrType))
                {
                    var parameter = Expression.Parameter(entityType.ClrType, "p");
                    var filter = Expression.Lambda(
                        Expression.Equal(
                            Expression.Property(parameter, "IsDeleted"),
                            Expression.Constant(false)),
                        parameter);

                    modelBuilder.Entity(entityType.ClrType).HasQueryFilter(filter);
                }
            }
        }
    }
}
```

## Query Patterns

### Using Specifications

```csharp
// ✅ Simple query with specification
var activeEmployeesSpec = new ActiveEmployeesByOrganizationSpecification(orgId);
var employees = await _employeeRepository.ListAsync(activeEmployeesSpec);

// ✅ Paginated query
var pageSpec = new EmployeesPagedSpecification(orgId, pageNumber: 1, pageSize: 10);
var employees = await _employeeRepository.ListAsync(pageSpec);

// ✅ Count with specification
var count = await _employeeRepository.CountAsync(pageSpec);
```

### Avoiding N+1 Queries

```csharp
// ❌ DON'T: N+1 problem
var employees = await _context.Employees.ToListAsync();
foreach (var emp in employees)
{
    var org = emp.Organization;  // ❌ Separate query per employee!
}

// ✅ DO: Include related data
var spec = new EmployeeSpecification()
    .WithOrganization();  // .Include()

var employees = await _employeeRepository.ListAsync(spec);
foreach (var emp in employees)
{
    var org = emp.Organization;  // ✅ No additional queries
}
```

### Complex Filtering

```csharp
// ✅ Specification encapsulates complex logic
public class ElectionsBetweenDatesSpecification : Specification<Election>
{
    public ElectionsBetweenDatesSpecification(
        int organizationId,
        DateTime startDate,
        DateTime endDate)
    {
        Query
            .Where(e => e.OrganizationId == organizationId
                && e.StartDate >= startDate
                && e.EndDate <= endDate
                && e.Status == ElectionStatus.Active)
            .OrderByDescending(e => e.StartDate)
            .Include(e => e.Organization);
    }
}

// Usage:
var activeElectionsSpec = new ElectionsBetweenDatesSpecification(
    orgId,
    DateTime.UtcNow.AddDays(-30),
    DateTime.UtcNow.AddDays(30));

var elections = await _electionRepository.ListAsync(activeElectionsSpec);
```

## Performance Best Practices

### Indexing Strategy

```csharp
// ✅ Composite index for common queries
builder.HasIndex(e => new { e.OrganizationId, e.IsDeleted });

// ✅ Unique index
builder.HasIndex(o => o.ManagerEmail).IsUnique();

// ✅ Filtered index (only active records)
builder.HasIndex(e => e.Email)
    .HasFilter("[IsDeleted] = 0");
```

### Batch Operations

```csharp
// ✅ Batch insert many records
public async Task AddManyAsync(IEnumerable<Employee> employees)
{
    await _context.Employees.AddRangeAsync(employees);
    await _context.SaveChangesAsync();
}

// ✅ Batch delete with no-tracking for performance
public async Task SoftDeleteManyAsync(IEnumerable<int> employeeIds, int modifiedBy)
{
    var employees = await _context.Employees
        .Where(e => employeeIds.Contains(e.EmployeeId))
        .ToListAsync();

    foreach (var emp in employees)
    {
        emp.SoftDelete(modifiedBy);
    }

    await _context.SaveChangesAsync();
}
```

### Pagination for Large Result Sets

```csharp
// ✅ Always paginate when listing
public class PagedResult<T>
{
    public IEnumerable<T> Items { get; set; }
    public int Page { get; set; }
    public int PageSize { get; set; }
    public int Total { get; set; }
    public int TotalPages => (Total + PageSize - 1) / PageSize;
}

// ✅ In repository
public async Task<(IEnumerable<T>, int)> ListPagedAsync(
    int page = 1,
    int pageSize = 10,
    ISpecification<T> spec = null)
{
    const int maxPageSize = 100;
    pageSize = Math.Min(pageSize, maxPageSize);

    var total = spec == null
        ? await _dbSet.CountAsync()
        : await CountAsync(spec);

    var items = spec == null
        ? await _dbSet.Skip((page - 1) * pageSize).Take(pageSize).ToListAsync()
        : await ApplySpecification(spec)
            .Skip((page - 1) * pageSize)
            .Take(pageSize)
            .ToListAsync();

    return (items, total);
}
```

## Checklist for Database Changes

When modifying database schema:

- [ ] Update Domain entity first
- [ ] Create new migration: `dotnet ef migrations add {DescriptiveName}`
- [ ] Review generated migration SQL
- [ ] Add/update DbContext configuration in OnModelCreating
- [ ] Add appropriate indexes for filtered and foreign keys
- [ ] Update related specifications if query patterns change
- [ ] Update seed data if needed
- [ ] Run migrations locally: `dotnet ef database update`
- [ ] Tests still pass: `dotnet test`
- [ ] Commit both migration and entity changes together

## Common Mistakes to Avoid

❌ **Hard-delete without soft delete option**

```csharp
// ❌ DON'T: Permanent deletion
await _employeeRepository.DeleteAsync(employee);
```

✅ **DO: Use soft delete**

```csharp
// ✅ DO: Soft delete, recoverable
employee.SoftDelete(modifiedBy);
await _employeeRepository.UpdateAsync(employee);
```

---

❌ **No indexes on foreign keys**

```csharp
// ❌ DON'T: Missing indexes cause slow queries
public int OrganizationId { get; set; }
```

✅ **DO: Add indexes**

```csharp
// ✅ DO: Index on FK and common filter combos
builder.HasIndex(e => e.OrganizationId);
builder.HasIndex(e => new { e.OrganizationId, e.IsDeleted });
```

---

❌ **Querying without Include causes N+1**

```csharp
// ❌ DON'T: Additional query per item
var orgs = await _context.Organizations.ToListAsync();
foreach (var org in orgs)
{
    var employees = org.Employees;  // Separate query!
}
```

✅ **DO: Use Include or Specification**

```csharp
// ✅ DO: Single query with related data
var spec = new OrganizationWithEmployeesSpecification();
var orgs = await _orgRepository.ListAsync(spec);
```
